# Lista de Exercícios 2 - Parte 1

## POO em Python: ABC, Polimorfismo e Protocol

Este repositório reúne as soluções da Lista de Exercícios 2 - Parte 1, desenvolvidas em Python para a disciplina de Programação Orientada a Objetos.

O objetivo principal é demonstrar conceitos importantes da POO, como classes abstratas (ABC), herança, sobrescrita de métodos, polimorfismo, protocolos e duck typing.

---

# Organização do Projeto

O projeto está dividido em pastas por questões:

* `1_Midias_Educacionais`
* `2_Funcionarios_Empresa`
* `3_Notificacoes`
* `4_Impressao`
* `5_Armazenamento`

Cada pasta contém os arquivos relacionados à sua respectiva questão.

---

# Estrutura Utilizada

## Models

Representam as entidades principais do sistema, contendo atributos e comportamentos dos objetos.

Exemplos:

* Midia
* Funcionario
* Notificador
* Boleto
* Armazenador

## Main

Arquivos responsáveis pela execução e testes do sistema.

Neles são criados os objetos e demonstradas as funcionalidades das questões.

# Como Executar

Abra o terminal na pasta do projeto e execute o arquivo principal da questão desejada.


